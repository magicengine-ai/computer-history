# 计算机发展简史 - Computer History

一个科普计算机发展历史的静态网站，涵盖芯片、计算机、人工智能和科技公司的发展历程。

## 📖 项目简介

本网站旨在以通俗易懂的方式介绍：
- 🔌 **芯片是怎么来的？** - 从沙子到超级芯片的制造过程
- 🖥️ **计算机是怎么来的？** - 从机械计算到数字革命
- 🤖 **人工智能发展路径** - 从图灵测试到大语言模型
- 🏢 **科技公司历史** - IBM、苹果、微软、谷歌等巨头的发展历程
- 📅 **完整时间线** - 1940s-2020s 关键事件记录

